# –û—Å–Ω–æ–≤–Ω—ã–µ —Å—Ç—Ä–∞—Ç–µ–≥–∏–∏ –±–æ—Ä—å–±—ã —Å –∫–æ–ª–ª–∏–∑–∏—è–º–∏

## 1) Separate chaining (—Ü–µ–ø–æ—á–∫–∏)

–ö–∞–∂–¥—ã–π –±–∞–∫–µ—Ç ‚Äî —Å–ø–∏—Å–æ–∫ (–∏–ª–∏ –¥—Ä—É–≥–æ–π –∫–æ–Ω—Ç–µ–π–Ω–µ—Ä) —ç–ª–µ–º–µ–Ω—Ç–æ–≤, –ø–æ–ø–∞–≤—à–∏—Ö –≤ —ç—Ç–æ—Ç –±–∞–∫–µ—Ç.

–ü–ª—é—Å—ã:

* –ø—Ä–æ—Å—Ç–∞—è —Ä–µ–∞–ª–∏–∑–∞—Ü–∏—è;
* –ª–µ–≥–∫–æ —Ö—Ä–∞–Ω–∏—Ç—å –Ω–µ–æ–≥—Ä–∞–Ω–∏—á–µ–Ω–Ω–æ–µ —á–∏—Å–ª–æ —ç–ª–µ–º–µ–Ω—Ç–æ–≤;

## 2) Open addressing (–æ—Ç–∫—Ä—ã—Ç–∞—è –∞–¥—Ä–µ—Å–∞—Ü–∏—è)

–í—Å–µ —ç–ª–µ–º–µ–Ω—Ç—ã —Ö—Ä–∞–Ω—è—Ç—Å—è –Ω–µ–ø–æ—Å—Ä–µ–¥—Å—Ç–≤–µ–Ω–Ω–æ –≤ –º–∞—Å—Å–∏–≤–µ. –ü—Ä–∏ –∫–æ–ª–ª–∏–∑–∏–∏ –∏—â–µ—Ç—Å—è –¥—Ä—É–≥–∞—è –ø–æ–∑–∏—Ü–∏—è (probe sequence).

–û–±—â–∏–µ –≤–∏–¥—ã:

* **Linear probing** (–ª–∏–Ω–µ–π–Ω–æ–µ –ø—Ä–æ–±–∏—Ä–æ–≤–∞–Ω–∏–µ): `h_i = (h + i) % m`

  * –ø—Ä–æ—Å—Ç–∞—è, –Ω–æ —Å–∫–ª–æ–Ω–Ω–∞ –∫ –ø–µ—Ä–≤–∏—á–Ω–æ–π –∫–ª–∞—Å—Ç–µ—Ä–∏–∑–∞—Ü–∏–∏ (–¥–ª–∏–Ω–Ω—ã–µ –≥—Ä—É–ø–ø—ã).

  * –ª—É—á—à–µ–µ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ –ø—Ä–∏ –ø—Ä–∞–≤–∏–ª—å–Ω–æ –≤—ã–±—Ä–∞–Ω–Ω–æ–π `h2`, –º–∞–ª–æ—á–∞—Å—Ç–æ –∫–æ–ª–ª–∏–∑–∏–∏.

–ü–ª—é—Å—ã:

* –∫–æ–º–ø–∞–∫—Ç–Ω–µ–µ –ø–æ –ø–∞–º—è—Ç–∏ (–Ω–µ—Ç —É–∫–∞–∑–∞—Ç–µ–ª–µ–π);
* –ª—É—á—à–∏–π –∫–µ—à-–ª–æ–∫–∞–ª–∏—Ç–µ—Ç ‚Üí –±—ã—Å—Ç—Ä–µ–µ –Ω–∞ –ø—Ä–∞–∫—Ç–∏–∫–µ.

–ú–∏–Ω—É—Å—ã:

* —Å–ª–æ–∂–Ω–µ–µ —É–¥–∞–ª–µ–Ω–∏–µ (—Ç—Ä–µ–±—É–µ—Ç –º–∞—Ä–∫–µ—Ä–∞ tombstone –∏–ª–∏ —Ä–µ–æ—Ä–≥–∞–Ω–∏–∑–∞—Ü–∏–∏);

## 3) Cuckoo hashing

## 5) Perfect hashing

# –†–µ—Ö–µ—à–∏—Ä–æ–≤–∞–Ω–∏–µ (resize)

–ö–æ–≥–¥–∞ `Œ±` –ø—Ä–µ–≤—ã—à–∞–µ—Ç –ø–æ—Ä–æ–≥ (–Ω–∞–ø—Ä–∏–º–µ—Ä 0.7), —Ç–∞–±–ª–∏—Ü—É –æ–±—ã—á–Ω–æ **—É–¥–≤–∞–∏–≤–∞—é—Ç** (`m' = 2*m`) –∏ —Ä–µ–∏–Ω—Å–µ—Ä—Ç—è—Ç –≤—Å–µ —ç–ª–µ–º–µ–Ω—Ç—ã —Å –Ω–æ–≤–æ–π —Ö–µ—à-—Ñ—É–Ω–∫—Ü–∏–µ–π (–æ–±—ã—á–Ω–æ —Ç–æ—Ç –∂–µ `GetHashCode` –∏ `% m'`). –≠—Ç–æ –¥–æ—Ä–æ–≥–æ (O(n)), –Ω–æ –∞–º–æ—Ä—Ç–∏–∑–∏—Ä–æ–≤–∞–Ω–Ω–æ –≤—Å—Ç–∞–≤–∫–∞ –æ—Å—Ç–∞—ë—Ç—Å—è O(1).

# –ü—Ä–∞–∫—Ç–∏—á–µ—Å–∫–∏–µ —Å–æ–≤–µ—Ç—ã –ø–æ —Ö–µ—à-—Ñ—É–Ω–∫—Ü–∏—è–º

# –°—Ä–∞–≤–Ω–µ–Ω–∏–µ —Å—Ç—Ä–∞—Ç–µ–≥–∏–π (–∫—Ä–∞—Ç–∫–æ)

# –£–¥–∞–ª–µ–Ω–∏–µ –≤ open addressing

* –ò—Å–ø–æ–ª—å–∑—É–π –º–µ—Ç–∫—É `Deleted` (tombstone). –£–¥–∞–ª–µ–Ω–Ω—ã–π —Å–ª–æ—Ç –Ω–µ–ª—å–∑—è –ø—Ä–æ—Å—Ç–æ –ø–æ–º–µ—Ç–∏—Ç—å –ø—É—Å—Ç—ã–º, —Ç.–∫. –æ–Ω –º–æ–∂–µ—Ç –ø—Ä–µ—Ä—ã–≤–∞—Ç—å —Ü–µ–ø–æ—á–∫—É –ø–æ–∏—Å–∫–∞.

# –ß—Ç–æ –µ—â—ë —É—á–∏—Ç—ã–≤–∞—Ç—å
